# petri nets project
Assignment MM HK251<br>
Lần đầu tải git về máy tại: 'https://git-scm.com/install/'<br>
Sau đó để clone project về máy, ở terminal hoặc git bash dùng lệnh: <pre>git clone https://github.com/TamiaoNguyen/petri-net-project.git</pre><br>
Đợi clone về, sau đó để nhập vào project dùng lệnh <pre>cd petri-net-project</pre><br>

Vào VSCode >> File >> Open Project >> Chọn folder petri-net-project<br>
Nhập vào branch bằng cách: <pre>git fetch</pre> <pre>git checkout feature/tên-branch</pre><br>
Viết code xong có thể đẩy branch lên github: <pre>git push -u origin feature/tên-branch</pre><br>
